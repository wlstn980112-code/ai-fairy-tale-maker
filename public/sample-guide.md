# 🎨 샘플 사용 가이드

## 테스트용 샘플 텍스트

### 📔 데일리 성장 일기 예시

```
오늘 유치원에서 친구 민수가 울고 있었어요. 
우리 아이가 다가가서 자기가 좋아하는 장난감을 빌려줬어요.
민수는 웃으면서 고맙다고 말했답니다.
```

```
오늘 처음으로 자전거 보조바퀴를 떼고 탔어요.
처음엔 무서워서 주저했지만, 용기를 내서 도전했어요.
넘어지기도 했지만 다시 일어나서 계속 연습했어요.
```

### ✨ 창작 동화 예시

```
숲 속에 마법의 나무가 있대요.
그 나무를 찾아가는 모험 이야기를 만들어주세요.
```

```
하늘을 나는 고양이와 친구가 되어 구름 위를 여행하는 이야기
```

```
바닷속 인어 공주의 생일 파티에 초대받은 이야기를 만들어주세요.
```

### 🦸 사진 속 주인공 예시

```
우리 아이가 슈퍼히어로가 되어 위험에 빠진 도시를 구하는 이야기
```

```
요정 나라의 공주님이 되어 마법 지팡이로 모두를 행복하게 만드는 이야기
```

```
우주 비행사가 되어 달나라에 가서 외계인 친구들을 만나는 이야기
```

## 💡 좋은 결과를 얻는 팁

1. **구체적으로 작성하기**
   - ❌ "놀았어요"
   - ✅ "친구들과 공원에서 숨바꼭질을 했어요"

2. **감정 표현 포함하기**
   - ❌ "밥을 먹었어요"
   - ✅ "처음으로 피망을 용감하게 먹어봤어요"

3. **교육적 포인트 암시하기**
   - 나눔, 용기, 친절, 인내, 도전, 배려 등

4. **사진과 연결하기**
   - 사진 속 옷차림이나 장소를 언급하면 더 현실감 있는 동화가 만들어져요

## 🎯 각 모드별 특징

### 데일리 성장 일기
- 실제로 일어난 일 기록
- 교육적 가치 강조
- 칭찬과 격려 중심

### 창작 동화
- 완전한 판타지 스토리
- 기승전결이 명확한 구조
- 상상력 자극

### 사진 속 주인공
- 아이의 실제 모습 반영
- 아이가 주인공인 모험
- 자존감 향상 효과
